Creating a User

Open the application in your web browser.
Locate the "Create User" section.
Enter the user's name and  then click the "Create User" button.
The new user will appear in the dropdown menu for task assignment.
=========================================================================
Assigning a Task
Select a user from the dropdown menu.
Enter the task details, including the task name and due date.
Note that the due date must be a future date.
Click the "Add Task" button to add the task to the selected user.
=========================================================================
Viewing and Managing Tasks
Tasks assigned to users will be stored in Local Storage.
Tasks can be viewed and managed through the user interface.
Tasks will be labeled based on their completion status:
Done: If completed before the due date.
Overdue: If completed after the due date.
